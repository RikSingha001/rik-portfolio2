Rik Singha - My Programming Journey

From Stone-Paper-Scissor to Full Stack Dreams

How I Fell in Love with Code

Written by Rik Singha on July 2, 2025

Early Life & My First Code

I started my journey from scratch — with zero background in computer science. My curiosity led me to write my very first program, a Stone-Paper-Scissor game using JavaScript. That simple project sparked something in me that hasn't stopped growing.

Learning PHP & Building Logic

After JavaScript, I taught myself PHP. It gave me a clear idea of how servers work, how forms handle data, and how logic flows behind the scenes. I built dynamic forms, login systems, and even started connecting PHP to MySQL using `mysqli` and `PDO`.

Falling for Java & Backend

Recently, I’ve developed a deep interest in backend development using Java. I’ve been learning the fundamentals of Object-Oriented Programming, memory management, and preparing to master Spring Boot. I enjoy understanding how the backend powers real-world applications.

My Capstone Project

I recently completed a fully functional Cab Booking System using Next.js, JWT authentication, and a raw PHP backend. This system supports role-based login (User, Driver, Vendor), ride bookings with filters, and a full dashboard view. It was challenging, but extremely fulfilling.

I am now continuously pushing myself to become a proficient backend developer. I'm excited for what lies ahead and look forward to mastering Java and Spring Boot to build scalable systems.
